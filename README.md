## 🎨 Creative App in Action

Unlike analytical tools that crunch data, **BrandGPT performs original creative work:**

Input:  "Sustainable coffee shop for remote workers"

Output: 

  → Name: "Brew Haven" / "The Grind Collective"
 
  → Tagline: "Fuel Your Focus, Sustainably"
  
  → Pitch: "Where productivity meets purpose..."
  
  → Colors: Earthy greens + warm browns
  
  → Font: Modern Sans-Serif for tech appeal
  
**This is creativity at machine speed.** From idea to brand identity in seconds.

## 🚀 Run Locally

### Prerequisites
- Node.js v18+
- GitHub Models API token ([Get here](https://github.com/settings/tokens))

### Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/Nishakr23/BrandGPT.git
cd BrandGPT

# 2. Install dependencies
npm install

# 3. Setup environment
cp .env.example .env
# Now edit .env and add: GITHUB_TOKEN=your_token_here

# 4. Launch
npm start

# 5. Open http://localhost:3000

⚠️ Note for Judges: API tokens are not committed for security reasons - industry best practice. Full working demo is in the video submission above. All code is complete and reviewable. Use your own GitHub Models token to test locally.
