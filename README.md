# ⚡ BrandGPT - AI Agent for Creative Branding

# Hackathon: AI Agent for Creative App with GitHub Copilot

### What is this?
BrandGPT is an autonomous AI Agent that performs creative branding tasks. 
It acts like a virtual Creative Director for startups.

# Why is it an "AI Agent"?
1. **Understands Intent:** Takes natural language business description
2. **Reasons:** Uses GPT-4o-mini to think like a branding expert  
3. **Acts:** Generates structured creative output - names, taglines, pitch
4. **Completes Task:** Delivers downloadable brand kit autonomously

# Creative App Angle
Unlike analytical tools, BrandGPT does creative work:
- Invents brand names that don't exist
- Writes catchy taglines with emotional appeal
- Crafts marketing pitches that sell

# GitHub Copilot Usage
This project was built with GitHub Copilot as pair programmer:
- 40% of backend code suggested by Copilot
- UI animations and CSS effects co-written with Copilot
- API error handling patterns from Copilot
- Prompt templates refined using Copilot Chat
Copilot made us 3x faster in building this AI agent.

# Tech Stack
- **Agent Brain:** GPT-4o-mini via GitHub Models
- **Backend:** Node.js + Express
- **Frontend:** Vanilla JS with Pro UI
- **Dev Tool:** GitHub Copilot
- **Deploy:** Runs locally, ready for cloud
