## 🎨 Creative App in Action

Unlike analytical tools that crunch data, **BrandGPT performs original creative work with market context:**

**Input:**  "Sustainable coffee shop for remote workers"

**Process:**  
1. Foundry IQ retrieves: remote work trends, eco-cafe color psychology, competitor analysis
2. GPT-4o-mini reasons: "Needs focus + sustainability + community vibe"
3. Agent creates complete brand

**Output:** 
  → **Name:** "Brew Haven" / "The Grind Collective"
  → **Tagline:** "Fuel Your Focus, Sustainably"  
  → **Pitch:** "Where productivity meets purpose..."
  → **Colors:** Earthy greens + warm browns - grounded in biophilic design research
  → **Font:** Modern Sans-Serif for tech appeal
  → **Brand IQ: 92/100** - Scored using retrieved market benchmarks

**This is creativity at machine speed, grounded in reality.** From idea to market-ready brand in 10 seconds.

## 🚀 Run Locally

### Prerequisites
- Node.js v18+
- GitHub Models API token ([Get here](https://github.com/settings/tokens))
- Microsoft Foundry IQ access

### Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/Nishakr23/BrandGPT.git
cd BrandGPT

# 2. Install dependencies
npm install

# 3. Setup environment
cp .env.example .env
# Add: GITHUB_TOKEN=your_token_here
# Add: FOUNDRY_IQ_ENDPOINT=your_endpoint

# 4. Launch
npm start

# 5. Open http://localhost:3000
⚠️ Security Note: API tokens are not committed - industry best practice per hackathon rules. Full working demo in video. Use demo data only.
